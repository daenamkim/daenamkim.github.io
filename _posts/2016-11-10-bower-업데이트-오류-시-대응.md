#bower #angularjs

sudo gem install bundler -n /usr/local/bin/
sudo gem install bundler 로 할 경우 /usr/bin/ 에 설치가 되는데 이곳에는 퍼미션 오류가 발생함.

rbenv 를 설치할 경우 현재 쉘 유저를 그대로 사용하기 때문에 sudo 없이 그냥
gem install bundler 를 하면 됨.
